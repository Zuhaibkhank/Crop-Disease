# 🌿 Crop Disease Detection System

## 📌 Overview

This project is a **Machine Learning based web application** that detects crop diseases from leaf images.
It helps farmers identify diseases early and take preventive actions.

---

## 🚀 Features

* 🌱 Detect crop diseases using image input
* 📊 Machine Learning model (CNN)
* 🌐 Web interface using Flask
* ⚡ Fast and accurate prediction
* 🧑‍🌾 Helpful for farmers

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python (Flask)
* **ML Model:** CNN (PyTorch / TensorFlow – jo use kiya ho)
* **Libraries:** NumPy, Pandas, OpenCV

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/zuhaibkhank/Crop-Disease.git

# Go to project folder
cd Crop-Disease

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

---

## 📊 How It Works

1. User uploads a leaf image
2. Model processes the image
3. Disease is predicted
4. Result displayed on screen
